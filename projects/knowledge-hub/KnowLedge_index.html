HR Knowledge Hub

Building a structured HR knowledge system to replace fragmented experience-based operations.

HTML / CSS Knowledge Management System Design

Context

In HR operations, knowledge is often scattered across individuals, documents, and informal communication. When key people leave or projects hand over, critical information is lost or requires significant effort to reconstruct.

Problem

  • HR project knowledge depends on individuals
  • SOPs are fragmented and hard to navigate
  • New HR members require long onboarding time
  • No centralized system for process visibility

Solution

Designed and built a static HTML-based knowledge platform that organizes HR processes, project timelines, system logic, and templates into a structured, easy-to-navigate system.

System Design

Key design thinking:

  • Project-based navigation (Performance / Promotion)
  • Role-based perspective (HRBP / HRIS)
  • Timeline-driven SOP structure
  • Modular page design for scalability

Impact

  • Reduced onboarding dependency on senior HR
  • Improved knowledge accessibility
  • Standardized HR project execution framework

Reflection

This project shifted my perspective from executing HR processes to designing systems. Instead of focusing on completing tasks, I began focusing on building structures that allow others to execute consistently without relying on tacit knowledge.
